### Why Pi4J?

Pi4J provides **a friendly, object-oriented I/O API and implementation libraries for Java programmers** to access the **full I/O capabilities of the Raspberry Pi platform**. It abstracts away the low-level native integration and interrupt monitoring, so you can **focus on your application's business logic**.

Ready to dive in?

* [Prepare a Raspberry Pi for Java development](https://www.pi4j.com/prepare/)
* Starter example: [a single-file application with JBang](/examples/jbang/jbang_minimal_example)
* For experienced Java developers: [an application with Maven or Gradle](/getting-started/minimal-example-application)

Pi4J has been helping Java developers blink LEDs, read sensors, build kiosk interfaces, and control robots since **2012**. In February 2026 the project joined the [Commonhaus Foundation](https://www.commonhaus.org/) to secure its long-term future. Learn more about the project's [history and versions](/about/history/).
